# firechat-vue

Projeto para estudos de chat em tempo real. Desenvolvido com VueJS e Firebase.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### To deploy
```
firebase deploy
```
